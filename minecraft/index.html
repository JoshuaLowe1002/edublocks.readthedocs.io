    What is Minecraft Pi Edition?

    Minecraft Pi Edition is a version of Minecraft developed for the Raspberry Pi. It is based on an old version of Pocket Edition but is slightly cut down, containing a revised feature set and support for multiple programming languages. EduBlocks has a feature that includes the Minecraft Python Library to interact with the Minecraft world.

    Starting Minecraft Pi Edition

    Minecraft Pi can easily be started from the Raspberry Pi Menu. Simply click on the Raspberry Pi logo in the top left hand corner of your Raspberry Pi desktop and navigate to Games and click on Minecraft Pi.

    This may not work on your VNC Servers! We recommend using a standard HDMI setup.

    Command Reference

    The below commands are those that can be found in EduBlocks. The block will be shown, and below it will be the Python code and then a breif description.

    1. Import Minecraft

    Python Code:

    from mcpi.minecraft import Minecraft

    Usage: This command imports the Minecraft API for us to interact with Minecraft via Python & EduBlocks

    2. MCPI Block Import

    Python Code:

    from mcpi import block

    Usage: This command allows us to reference to the Minecraft Blocks via their names rather than the numbered id's
